<h1 align="center">Frame</h1>

<p align="center">
  一个安静的 macOS 工具，用来截图、录屏，并找回刚刚捕获的内容。
</p>

<p align="center">
  <a href="README.md">English</a>
</p>

Frame 适合那些需要快速展示内容的小场景：反馈问题、解释想法、保存设置、标记界面，或录下一段短操作。按下快捷键，选择你需要的内容，然后复制、保存、编辑、钉住，或稍后再找回来。

所有内容都在你的 Mac 上处理。Frame 不需要账号，也不会上传截图或录屏。

## 用起来是什么感觉

Frame 常驻菜单栏，不会打开一个很大的主窗口。每次截图或录屏后，屏幕角落会出现一个小预览。你可以直接复制、保存、编辑、让它保持可见，或者关掉它。

目标很简单：拿到你需要的内容，然后继续手头的事。

## 可以做什么

### 截图

- 用 `Command+Shift+A` 选择一块区域。
- 捕获一个窗口。
- 通过滚动捕获较长的内容。
- 支持多显示器。
- 将截图保存为 PNG 文件。

### 从预览继续处理

- 复制截图或录屏。
- 保存到你选择的文件夹。
- 把截图拖到其他应用里。
- 查看更大的预览。
- 钉住图片，让它在工作时保持可见。

### 快速编辑

- 添加文字、箭头、形状、高亮、画笔标记，或给敏感内容打马赛克。
- 添加后可以移动。
- 调整大小，或直接删除。
- 通过按上下文显示的顶部样式控件调整颜色和大小。
- 使用画布快捷键，或双击对象回到对应编辑工具。
- 撤销和重做。
- 保存编辑后的结果。

### 录屏

- 将选定区域录制为 MP4 或 GIF。
- 选择是否显示鼠标、点击和按键。
- 停止、重新开始或删除录屏。
- 裁剪 MP4，并导出更快或更慢的版本。

### 找回最近内容

- 打开最近的截图和录屏。
- 把之前的内容放回角落预览里。
- 自己决定最近内容保留多久。

## 权限

Frame 需要 macOS 的屏幕录制权限（Screen Recording），这样才能在截图或录屏时看到屏幕上的内容。这个名字听起来像只和录屏有关，但截图类应用也需要它。

更多细节见 [macOS Permissions](docs/permissions.md)。
