Menu bar icons for Frame

Two sets — pick one (or ship both).

1) FrameStatusIconTemplate.png (+ @2x, @3x)
   - Pure black, transparent background.
   - Use as a macOS template image: set image.isTemplate = true
     (the "Template" suffix in the filename also signals this).
   - macOS tints automatically: dark in light mode, white in dark mode,
     and white-on-blue when the status item is highlighted.
   - Recommended.

2) FrameStatusIcon.png (+ @2x, @3x)
   - Black outline + handles with the mint accent (#00b894) preserved.
   - Use only if you want the brand color visible regardless of system tinting.
   - NOT a template image — set image.isTemplate = false.

Base point size: 18×18.  Files at 1×=18px, 2×=36px, 3×=54px.
